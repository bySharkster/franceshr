## Button Variants

These colors are applied through the shadcn/ui button component variants:

- **Default**: Uses Primary color with white text
- **Destructive**: Uses Destructive color for dangerous actions
- **Outline**: Uses Border color with Background fill
- **Secondary**: Uses Secondary color
- **Ghost**: Transparent with Accent hover state
- **Link**: Uses Primary color with underline
