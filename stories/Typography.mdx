# Typography

The FrancesHR application uses a clean, modern typography system based on SF Pro Display for optimal readability across all platforms.

## Font Families

We use three primary font stacks:

### Sans Serif (Default)

**Primary font family for body text, headings, and UI elements**

```css
font-family:
  "SF Pro Display",
  -apple-system,
  BlinkMacSystemFont,
  "Segoe UI",
  Roboto,
  Oxygen,
  Ubuntu,
  Cantarell,
  "Open Sans",
  "Helvetica Neue",
  sans-serif;
```

### Monospace

**Used for code snippets and technical content**

```css
font-family: "SF Mono", monospace;
```

## Usage Guidelines

### Headings

- Use **H1** for page titles (one per page)
- Use **H2** for major section headings
- Use **H3-H6** for subsections and nested content
- Keep heading hierarchy logical and sequential

### Body Text

- **16px** for main content and descriptions
- **14px** for form labels, buttons, and secondary UI text
- **12px** for captions, timestamps, and helper text

### Font Weights

- **700 (Bold)**: Main headings and emphasis
- **600 (Semibold)**: Subheadings and important UI elements
- **500 (Medium)**: Buttons, labels, and medium emphasis
- **400 (Regular)**: Body text and default content

### Best Practices

- Maintain consistent line heights for readability
- Use letter spacing adjustments for all-caps text
- Ensure sufficient contrast between text and background
- Test typography at different screen sizes
